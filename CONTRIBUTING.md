# Contributing

## Reporting Issues
- Check existing issues first
- Include OS and VS Code version
- Include error output

## Adding Tool Categories
1. Update `CATEGORIES` dict in `copilot-tools.py`
2. Add to SKILL.md tool reference
3. Add to README.md table

## Submitting Changes
1. Fork the repository
2. Create feature branch
3. Test locally
4. Submit PR
